# shop
项目简介：
模仿天猫的简易版购物网站，使用springmvc+mybatis构建，前端框架使用jquery，前端页面使用纯css、js构建，用redis存储用户cookie信息<br>
实现购物车，订单，排序，搜索，注册，登录，权限拦截，cookie功能<br>
